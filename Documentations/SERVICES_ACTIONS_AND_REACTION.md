# Services actions and reactions

&ensp; *For each service, we defined actions (events that trigger a workflow) and reactions (tasks the platform can perform in response).*

## Dropbox

&ensp; *A file hosting site offering storage as a service and online file sharing*

#### Actions :

&emsp; ●  **Add a file** **~**  happens when a file is added to a Dropbox folder

#### Reactions :

&emsp; ● **Upload a file** **~** upload a file from the platform to a Dropbox folder
<br>
## Github
&ensp; *A platform that allows developers to create, store, manage, and share their code*

#### Actions :

&emsp; ● **New push to a repository** **~** happens when the user submits changes to a repository

#### Reactions :

&emsp; ● **Create an issue** **~** create a new issue in a repository where the user is
<br>
## Gmail

&ensp; *Google’s messaging system*

#### Actions :

&emsp; ● **New email received** **~** happens when any new email arrives in the box

&emsp; ● **New important email received** **~** happens when a new email marked as “Important” arrives in the box

#### Reactions :

&emsp; ● **Send an email** **~** sends an email with the connected Gmail account
<br>

## Microsoft

&ensp; *An American computer and microinformatics multinational*

#### Actions :

&emsp; ● **New mail received** **~** happens when a new email arrives in the OUtlook box

&emsp; ● **New file in OneDrive** **~** happens when a new file is added to a OneDrive folder

#### Reactions :

&emsp; ● **Send mail** **~** sends a new email with Outlook

&emsp; ● **Upload file to OneDrive** **~** Uploads a file in a OneDrive folder

<br>

## Notion

&ensp; *An all-in-one productivity app* for writing, planning, and organizing notes, projects, tasks, and more

#### Actions :

&emsp; ● **User is mentioned** **~** happens when the connected user gets mentioned in any pages or spaces

&emsp; ● **Page is created** **~** happens when a new page is created in the user’s workspace

#### Reactions :

&emsp; ● **Create a page** **~** creates a new page in the user’s workspace

<br>

## Twitch

&ensp; *An interactive livestreaming service for content spanning gaming, entertainment, sports, music, and more*

#### Actions :

&emsp; ● **Streamer goes live** **~** happens when the user favorite streamer starts a new live

#### Reactions :

&emsp; ● **Suggest similar streamers** **~** suggest to the user streamers that might interest him based on what the user watches or likes
